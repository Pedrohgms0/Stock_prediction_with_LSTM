# Stock_prediction_with_LSTM
Modelo LSTM multivariado para previsão de retorno diário em uma carteira de energia da NYSE (XOM, CVX, SLB, HAL). Treinado em 2010–2019, testado em 2023–2025 com +22pp de alpha sobre buy &amp; hold e 90% de consistência em 50 seeds independentes. PyTorch · pandas-ta · threshold dinâmico · análise de robustez.
